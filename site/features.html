KiCAD Features

KiCAD is a powerful and FOSS PCB Design CAD software. There are a few main apps that come installed with KiCAD.

  • Schematic Capture

    KiCAD uses a hierarchical sheet scheme. Each project can have multiple sheets and you can connect parts of one sheet to another. KiCAD offers an impressive symbol library with thousands of common parts, and additional ones can be imported from Ultra Librarian / SnapEDA. An easy-to-use symbol designer is available as well.

  • PCB Layout

    KiCAD is a one project, one-PCB model. You can quickly update PCBs from schematic with one button. By default, powerful tools like Interactive Router prevent unintentional shorts. Unlike free versions of proprietary software, there are no limits to PCB board size, and supports up to 16 copper layers. KiCAD is highly customizable!

  • 3D Viewer

    Without opening any additional apps, a built-in PCB viewer allows you to generate 2D and 3D models of your PCB. You can import custom models or use pre-installed bundled models. KiCAD also gives you raytracing support for better boards, and the ability to customize colors and orientation to your heart's content.
